# Java Docker App 🐳

A Spring Boot REST API containerized with Docker using multi-stage builds.

## 🛠️ Tech Stack
- Java 17 + Spring Boot 3.2
- Docker (multi-stage build)
- Maven

## 🚀 Run Locally
docker pull mnqw1209/java-docker-app:v1
docker run -d -p 8080:8080 mnqw1209/java-docker-app:v1

Open: http://localhost:8080
